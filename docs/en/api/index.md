# API Overview

This section covers the Industry Vis API interfaces.

## Tauri Commands

The application uses Tauri IPC for frontend-backend communication. See [Tauri Commands](/en/api/commands).

## Data Models

Data structure definitions used by the system. See [Data Models](/en/api/models).

## Command List

| Command | Description |
|---------|-------------|
| `load_config` | Load configuration file |
| `save_config` | Save configuration file |
| `test_connection` | Test database connection |
| `get_available_tags` | Get available tags list |
| `query_history` | Query history data |
| `export_to_csv` | Export data to CSV |
