# 数据查询

## 时间范围选择

系统提供两种时间范围选择方式：

### 预设时间范围

- **最近1小时** - 查询过去1小时的数据
- **最近6小时** - 查询过去6小时的数据
- **最近24小时** - 查询过去24小时的数据（默认）
- **最近7天** - 查询过去7天的数据

### 自定义时间范围

选择"自定义"后，可以通过日期时间选择器精确指定开始和结束时间。

## 标签筛选

### 标签列表

系统会自动从数据库获取所有可用的标签名称。

### 多选

支持同时选择多个标签进行查询和对比。

### 搜索

在标签选择器中输入关键字，可以快速过滤标签列表。

## 查询执行

1. 选择时间范围
2. 选择要查询的标签（可选，不选则查询所有标签）
3. 数据会自动加载并显示

## 数据表格

查询结果以表格形式展示：

| 列 | 说明 |
|----|------|
| 时间 | 数据记录的时间戳 |
| 标签名 | 数据点的标签名称 |
| 数值 | 测量值 |
| 质量 | 数据质量标志 |

### 表格功能

- **排序** - 点击列标题进行排序
- **筛选** - 标签列支持筛选
- **分页** - 默认每页50条记录
- **虚拟滚动** - 高效处理大量数据

## 刷新数据

点击"刷新"按钮重新加载数据。
