# 数据导出

## CSV 导出

系统支持将查询结果导出为 CSV 格式文件。

### 导出步骤

1. 执行数据查询
2. 点击"导出 CSV"按钮
3. 在弹出的文件保存对话框中选择保存位置
4. 输入文件名并保存

### CSV 格式

导出的 CSV 文件包含以下列：

```csv
DateTime,TagName,TagVal,TagQuality
2024-01-01T00:00:00.000,温度传感器1,25.5,良好
2024-01-01T00:00:01.000,温度传感器1,25.6,良好
...
```

| 列名 | 说明 |
|------|------|
| DateTime | ISO 8601 格式的时间戳 |
| TagName | 标签名称 |
| TagVal | 数值（保留4位小数） |
| TagQuality | 质量标志 |

### 注意事项

- 导出的是当前筛选后的数据
- 如果数据量很大，导出可能需要一些时间
- 文件使用 UTF-8 编码
