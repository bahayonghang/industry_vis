# 简介

Industry Vis 是一个基于 Tauri 的工业数据查看系统，专为读取、处理和展示工业运行数据而设计。

## 主要功能

- **SQL Server 连接** - 支持连接 SQL Server 数据库，读取工业历史数据
- **时间范围选择** - 预设时间范围（1小时/6小时/24小时/7天）和自定义时间范围
- **标签筛选** - 多选标签，支持搜索过滤
- **数据表格** - 虚拟滚动表格，显示时间、标签名、数值、质量标志
- **折线图** - 多系列折线图，支持缩放平移和数据点提示
- **CSV 导出** - 支持导出筛选后的数据为 CSV 文件

## 数据模型

系统默认查询 `控制器数据库` 的 `历史表`，表结构如下：

| 列名 | 类型 | 说明 |
|------|------|------|
| DateTime | datetime | 时间戳 |
| TagName | nchar(50) | 标签名称 |
| TagVal | real | 数值 |
| TagQuality | nchar(10) | 质量标志 |
